## Edgar Magdaleno Test

## Dependencies
Mongo
Node

## Set up the client
From project root:
- cd client
- npm install
- npm run build

## Set up the server
From project root:
- cd server
- npm install
- node app.js

## Set up the database
Make sure Mongo is installed and exposed at 127.0.0.1, default port.

## Run
By running de server you expose the app at http://localhost:80/
